# JVM이란?
> Java Virtual Machine : 자바를 실행하기 위한 가상 기계(컴퓨터)

## 정의
 - Java 는 OS에 종속적이지 않다는 특징
 - OS에 종속받지 않고 실행되기 위해선 OS 위에서 Java 를 실행시킬 무언가가 필요
  => JVM
 - 즉, OS에 종속받지 않고 CPU 가 Java를 인식, 실행할 수 있게 하는 가상 컴퓨터
